This project (aka Miniblog) written on Django and demonstrating basic principles in working with Django's Classes, Views, Templates and Tests.

This project was written as the final exam work after my employer's internal Django education.

Project utilizes Django's internal users' administration, and allows to write simple posts, optionally commented by other users.

There is no complicated things like JS scripts, this project in most of its parts focused on backend.

You are free to go starting with user creation in admin panel. Then you can start posting some posts and commenting them under different users.

Users can be divided into groups by their ability to create posts. This relies on internal Django's permission system: the easiest way is to create user groups in admin panel with or without miniblog.add_post permission to allow specific user to comment only.

API interface is available here. Neither updating nor creating nor deleting objects is supported for now.
